关于仪器检测
可检测以下信息

1、开关状态. 2、指示灯状态. 3、数字仪表读数. 4、指针式仪表读数.5、二维码信息

关于图像拼接
使用说明如下

在需要拍摄的物体过大时，可将其分块拍摄为若干张照片，每两张相邻照片保留一定重叠部分，将这些图片按顺序（从左往右或从上往下）传入即可拼接一张完整的长图。（本服务提供上下图像拼接和左右图像拼接两种方式）


关于人脸识别
可实现功能

用户将照片传入，本服务可标注出人脸并比对人脸库中的人脸信息，判断是否在此人信息是否在数据库中，如在册即给出此人身份信息

关于添加模型
使用说明如下

可在服务中选择需要训练的类型，并选择“添加图片”与“添加xml”上传照片和xml文件（注意:图片名和xml名须一致，保证图片与对应的xml一一对应），选择开始训练即可，训练结束后将更新模型
